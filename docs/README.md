# ハンズオンドキュメント

シミュレーションゲーム作成ハンズオンへようこそ!

このハンズオンでは、単語を教えると話してくれる動物キャラクターのWebアプリケーションを、段階的に作成していきます。

## 学習の流れ

このハンズオンは4つの章で構成されています。第1章から順番に進めてください。

### 第1章: 基礎練習

Web開発の基礎を練習します。HTML、CSS、JavaScriptの基本から、3D表示まで学びます。

[第1章: 基礎練習へ](./1-practice/README.md)

**学習内容:**
- HTML と CSS の基本
- JavaScriptのデバッグ方法
- ダイアログ（入力画面）の作り方
- 時間を扱う処理（setTimeout）
- 非同期処理（async/await）
- Three.jsを使った3D表示

---

### 第2章: UI構築

3Dキャラクターと対話できるUIを構築します。第1章で学んだことを活用して、実際のアプリケーションを作ります。

[第2章: UI構築へ](./2-ui/README.md)

**学習内容:**
- 3Dキャラクターの表示
- メニューとダイアログの実装
- 単語を教える機能
- キャラクターが話す機能
- GitHubでのコード管理
- Vercelへのデプロイ

---

### 第3章: API構築

データをサーバーで管理できるようにします。バックエンドAPIとデータベースを追加します。

[第3章: API構築へ](./3-api/README.md)

**学習内容:**
- Vercel CLIの使い方
- APIエンドポイントの作成
- Supabaseデータベースの設定
- データベースとの連携
- 環境変数の管理

---

### 第4章: AI統合

Google Gemini AIを使って、キャラクターに性格を持たせます。

[第4章: AI統合へ](./4-ai/README.md)

**学習内容:**
- セリフ生成APIの実装
- Google Gemini AIの統合
- プロンプトエンジニアリング

---

## 必要な準備

### 開発環境

- **テキストエディタ** - [VS Code](https://code.visualstudio.com/) を推奨
- **Webブラウザ** - Chrome または Firefox の最新版
- **ターミナル** - コマンドを入力できる環境

### アカウント

- **GitHub** (第2章で使用) - https://github.com/
- **Vercel** (第2章で使用) - https://vercel.com/
- **Supabase** (第3章で使用) - https://supabase.com/
- **Google AI Studio** (第4章で使用) - https://aistudio.google.com/

## プロジェクトの構成

```
docs/           # このハンズオンドキュメント
├── 1-practice/ # 第1章: 基礎練習
├── 2-ui/       # 第2章: UI構築
├── 3-api/      # 第3章: API構築
└── 4-ai/       # 第4章: AI統合
```
