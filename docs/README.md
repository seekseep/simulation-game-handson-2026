# ハンズオンドキュメント

シミュレーションゲーム作成ハンズオンへようこそ!

このハンズオンでは、単語を教えると話してくれる動物キャラクターのWebアプリケーションを、段階的に作成していきます。

## 学習の流れ

このハンズオンは4つの章で構成されています。第1章から順番に進めてください。

### 第1章: 基礎練習

Web開発の基礎を練習します。HTML、CSS、JavaScriptの基本から、3D表示まで学びます。

[第1章: 基礎練習へ](./1-practice/README.md)

**学習内容:**
- HTML と CSS の基本
- JavaScriptのデバッグ方法
- ダイアログ（入力画面）の作り方
- 時間を扱う処理（setTimeout）
- 非同期処理（async/await）
- Three.jsを使った3D表示

**所要時間:** 約2〜3時間

---

### 第2章: UI構築

3Dキャラクターと対話できるUIを構築します。第1章で学んだことを活用して、実際のアプリケーションを作ります。

[第2章: UI構築へ](./2-ui/README.md)

**学習内容:**
- 3Dキャラクターの表示
- メニューとダイアログの実装

**所要時間:** 約3〜4時間

---

### 第3章: API構築

データをサーバーで管理できるようにします。バックエンドAPIとデータベースを追加します。

[第3章: API構築へ](./3-api/README.md)

**学習内容:**

**所要時間:** 約3〜4時間

### 第4章: AI統合

Google Gemini AIを使って、キャラクターに性格を持たせます。

[第4章: AI統合へ](./4-ai/README.md)

## 全体の学習時間

合計: 約9〜13時間

一度に全部やる必要はありません。自分のペースで進めてください。

## 学習の進め方

### 推奨の進め方

1. **第1章から順番に進める** - 各章は前の章の内容を前提としています
2. **コードは自分で打ち込む** - コピー&ペーストだけでなく、実際に打ち込むことで理解が深まります
3. **動作確認を必ず行う** - 各ステップで動作を確認してから次に進みましょう
4. **わからなくても進める** - 最初は理解できなくても、後で「あ、そういうことか!」となることがよくあります

### 学習のヒント

- **エラーを恐れない** - エラーは学習の一部です。エラーメッセージをよく読んで、何が問題なのか考えましょう
- **検索を活用する** - わからない用語やエラーメッセージは検索してみましょう
- **休憩を取る** - 集中力が切れたら休憩しましょう。休憩後に見ると理解できることもあります

## 必要な準備

### 開発環境

- **テキストエディタ** - [VS Code](https://code.visualstudio.com/) を推奨
- **Webブラウザ** - Chrome または Firefox の最新版
- **ターミナル** - コマンドを入力できる環境

### アカウント（すべて無料）

各章を始める前に作成すれば大丈夫です。

- **GitHub** (第2章で使用) - https://github.com/
- **Vercel** (第2章で使用) - https://vercel.com/
- **Supabase** (第3章で使用) - https://supabase.com/
- **Google AI Studio** (第4章で使用) - https://aistudio.google.com/

詳しくは [DOCUMENT_ARCHITECTURE.md](../.claude/DOCUMENT_ARCHITECTURE.md) をご覧ください。

## 困ったときは

- **エラーが出た** - エラーメッセージをよく読んで、何が問題なのか確認しましょう
- **動かない** - 各章の「トラブルシューティング」セクションを確認してください
- **理解できない** - まずは手を動かしてみましょう。後で理解できることもあります
