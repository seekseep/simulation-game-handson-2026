# 第2章: UI構築

## この章について

第2章では、3Dキャラクターと対話できるUIを構築します。

第1章で学んだHTML、CSS、JavaScript、Three.jsを活用して、実際のアプリケーションを作っていきます。
この章では一つのプロジェクトとして開発し、最終的にインターネット上に公開します。

UIの実装からロジックの追加、そしてGitHubでのコード管理とVercelでのデプロイまで、段階的に進めていきます。
セクション1と2では合計16個の動作する実装例を提供しているので、各ステップで自分のコードと比較しながら学習を進めることができます。

## 学習内容

この章では、以下のことを学びます:

- Three.jsを使った3Dキャラクターの表示
- HTMLダイアログ要素（`<dialog>`）を使ったUI実装
- メニュー、ダイアログなどのUIコンポーネントの作成
- JavaScriptモジュール（import/export）の使い方
- DOMの動的操作（createElement、appendChild）
- イベントリスナーと非同期処理（async/await）
- 配列とオブジェクトを使ったデータ管理
- テンプレートシステムによる発話機能の実装
- 配列メソッド（filter、forEach）の活用
- 一文字ずつ表示するアニメーション効果
- localStorageを使ったデータの永続化
- JSONによるデータのシリアライズ/デシリアライズ
- Gitの基本概念（コミット、プッシュ、プル）
- GitHubを使ったコード管理
- Vercelを使った本番環境へのデプロイ

## 前提条件

- 第1章を完了していること

GitHubとVercelのアカウントは、この章の途中で作成する手順を説明します。

## この章で作るもの

この章では、以下の機能を持つアプリケーションを作ります。

セクション1（ビュー作成）では10段階、セクション2（ロジック実装）では6段階の実装例を用意しているので、段階的に理解を深めながら進めることができます。

1. **3Dキャラクターの表示** - 動物のキャラクターが3Dで表示されます
2. **メニュー** - 「教える」「話す」ボタンでキャラクターと対話できます
3. **単語を教える** - キャラクターに新しい単語を教えることができます
4. **カテゴリー分け** - 単語を「あいさつ」「食べ物」「必殺技」「場所」に分類します
5. **キャラクターが話す** - 教えた単語を使って、キャラクターが話します
6. **アニメーション** - キャラクターが動きます
7. **データ保存** - ブラウザに単語を保存します（localStorage）

シンプルなUIから始めて、徐々に機能を追加していき、最終的にはGitHubでコードを管理してVercelでインターネット上に公開します。

## 章の構成

この章は以下の4つの節で構成されています。順番に進めてください。

1. [ビュー作成](1-view/README.md) - UIコンポーネントの実装方法を学びます。メニュー、ダイアログ、発話表示などの画面要素を10段階に分けて段階的に作成します。HTMLダイアログ要素やDOM操作の基本を習得できます。
2. [ロジック実装](2-logic/README.md) - アプリケーションの動作を実装します。配列やオブジェクトを使ったデータ管理、テンプレートシステムによる発話機能、localStorageでのデータ永続化を6つのステップで学びます。
3. [GitHub設定](3-github/README.md) - Gitとバージョン管理の基本を学びます。GitHubアカウントの作成、リポジトリの作成、VS Codeを使ったコミットとプッシュの方法を習得します。
4. [Vercelデプロイ](4-vercel/README.md) - 作成したアプリケーションをインターネット上に公開します。Vercelアカウントの作成から、GitHubとの連携、自動デプロイの設定まで行います。
