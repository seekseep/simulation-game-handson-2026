# シミュレーションゲーム作成ハンズオン 2026

単語を教えると話してくれる動物キャラクターを作る Web アプリケーション開発チュートリアルです。

## このプロジェクトについて

このプロジェクトは、プログラミング初心者が Web アプリケーション開発を学ぶための教材です。
3D グラフィックス、API、データベース、AI を組み合わせた実践的なアプリケーションを、段階的に作成していきます。

### 完成するアプリケーション

- 3D で表示された動物キャラクターに単語を教えることができます
- 教えた単語を使って、キャラクターが話します
- 単語はカテゴリー（あいさつ、食べ物、必殺技、場所）で管理されます
- AI がキャラクターの性格に合わせてセリフを生成します
- キャラクターがセリフに合わせてアニメーションします

## デモ

完成したアプリケーションのデモは [demo](demo) ディレクトリにあります。

ローカルで動作を確認する場合:

```bash
cd demo
npx vercel dev
```

ブラウザで http://localhost:3000 にアクセスしてください。

## ハンズオン

段階的に学習しながらアプリケーションを作成したい方は、[ハンズオンドキュメント](docs/README.md) をご覧ください。

### 学習内容

- **第1章: 基礎練習** - HTML/CSS/JavaScript の基礎
- **第2章: UI 構築** - Three.js で 3D キャラクター UI を作成
- **第3章: API 構築** - バックエンド API とデータベースの統合
- **第4章: AI 統合** - 生成 AI でキャラクターに性格を持たせる

### 対象者

- プログラミング初心者の方
- Web アプリケーション開発を学びたい方
- 3D グラフィックスや AI に興味がある方

### 前提知識

- ファイルとフォルダの基本操作ができること
- テキストエディタでファイルを編集できること

HTML/CSS/JavaScript の知識がなくても、ハンズオンを通して学ぶことができます。

## 技術スタック

### バックエンド
- Vercel Serverless Functions (API)

- Git & GitHub

## 必要なアカウント

ハンズオンを進めるには、以下のアカウントが必要です（すべて無料プランで利用可能）:
